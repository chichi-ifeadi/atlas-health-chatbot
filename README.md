# Atlas Wellness Chatbot

Atlas is a personal wellness chatbot that helps users improve daily habits such as sleep, stress, energy, and routines. The system provides structured guidance while enforcing strict safety boundaries (no medical advice).

---

## Live Demo (Production)

https://your-railway-url.up.railway.app

---

## Features

- AI-powered wellness chatbot  
- Safety boundaries (no medical advice)  
- User authentication (login/signup)  
- Daily wellness check-ins  
- Dashboard with tracking  
- Streak feature (consecutive check-ins)  
- Email reminders (notifications)  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** Supabase  
- **APIs:** OpenAI, Resend  
- **Deployment:** Railway
