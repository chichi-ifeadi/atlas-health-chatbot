# AGENTS.md

## Goal
Build and maintain a clean, modern, professional app.

## UI rules
- Keep layouts balanced and symmetrical.
- Prefer reusable components.
- Maintain consistent spacing.
- Reduce unnecessary scrolling.
- Preserve functionality while improving UX.
- Keep the interface mobile-friendly when applicable.

## Code rules
- Do not add unnecessary files.
- Reuse existing components and styles when possible.
- Keep code modular and readable.
- Run linting after major changes.
